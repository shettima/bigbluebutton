# Introduction #

The BigBlueButton 0.81 server runs under Ubuntu 10.04 64-bit.  To setup your own BigBlueButton server:

  * [Install from packages](InstallationUbuntu.md) (Ubuntu 10.04 64-bit)
  * [Download BigBlueButton Virtual Machine](BigBlueButtonVM.md) (Mac OS X or Windows desktop)

After you've setup a BigBlueButton server, you can configure it using the `bbb-conf` tool, see

  * [BigBlueButton configuration tool](BBBConf.md)

If you are new to BigBlueButton, please read through the [overview](TechnicalOverview.md) pages to learn more about BigBlueButton's architecture and configuration.
