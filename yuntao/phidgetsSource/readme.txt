This submition is for using phidgets module within BBB.The souce code was update from revision 2579.

For connect phidget widgets, you have install the phidgets web service before run BBB web conference
you can find installer for your OS from here: http://www.phidgets.com/drivers.php 

After install the Phidget Control Panel successfully, please start web service with Port: 5001, Password: pass.
Then you are good to monitor RaiseHand Event with Phidget Servo and change slides with force and touch sensors.
