Red5Phone Readme

SIP Application for Standalone Red5 Server 0.8 RC2 is http://red5.4ng.net/sip.zip.

Unzip and move the sip folder to webapps.

You will need to install Red5 server 0.8.0 RC2.

Run red5phone application with http://your_red5_server:5080/sip

For help read http://code.google.com/p/bigbluebutton/wiki/Red5Phone
or
http://www.igniterealtime.org/community/thread/34414

To checkout SVN use http://red5phone.googlecode.com/svn/trunk/sip/
