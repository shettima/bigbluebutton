Classpublic class TestSuiteResult
InheritanceTestSuiteResult flash.events.EventDispatcher
ImplementsITestResultContainer

This class contains result information about the execution of a test suite. The TestMonitor class uses instances of this class to maintain state and display information for each TestSuite.
