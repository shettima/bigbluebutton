Interfacepublic interface ITestResultContainer extends ITestResult
ImplementorsTestCaseResult, TestSuiteResult

Interface implemented currently by TestSuiteResult, TestCaseResult. The interface guarantees the presence of the children and numberOfFailures properties, which indicates the hierarchy of the suites, cases and methods as well as indicate the number of methods that have failed.
