Classpublic class ChooseTestMethodResultEvent
InheritanceChooseTestMethodResultEvent flash.events.Event

A ChooseTestMethodResultEvent is fired by the TestProgressBar when the user clicks on a visual representation of a completed test. This event allows the TestResultDisplay code to select the appropriate result in the tree and display the stack trace information in the text area.
