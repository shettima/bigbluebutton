Packagenet.digitalprimates.fluint.monitor
Classpublic class TestMonitor
InheritanceTestMonitor flash.events.EventDispatcher

The TestMonitor class is a singleton that contains references to all SuitesResults, CaseResults and MethodResults in the system. TestCase calls methods within this class to report the results of individual test methods. The UI monitors the contents of this class to show a visualization to the user.
