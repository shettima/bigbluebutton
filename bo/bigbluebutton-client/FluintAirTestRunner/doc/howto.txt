about hout to run Ant+TestRunner_ant:

checkout new BBB-client
install adobe Air
install FlunitAirTestRunner-1.1.0.air(double-click, already checkin at BBB-client/FlunitAirTestRunner/other ) into C:\FluintAIRTestRunner
update BBB-client/FlunitAirTestRunner/build.xml(set testing-resort-output folder, swf folder)
build test-case(for example, UnitTestRunner_ant.mxml), and copy  UnitTestRunner_ant into BBB-client/FlunitAirTestRunner/swf/ folder
run BBB-client/FlunitAirTestRunner/build.xml
check testing resort at BBB-client/FlunitAirTestRunner/log

for example, one sample test is as following:

UnitTestRunner_ant.mxml(/src/ folder) - UnitTestSuite.as(/tests/unit/ folder) - LoginServiceTestCase.as(/tests/unit/org/bigbluebutton/modules/login/model/services/ folder)
build UnitTestRunner_ant.mxml to UnitTestRunner_ant.swf, copy to /FlunitAirTestRunner/swf/, then run builder.xml
