Welcome to BigBlueButton Developer Virtual Machine


This virtual machine (VM) is a fully working copy of BigBlueButton that you can try out BigBlueButton, the open source web conferencing system for distance learning.


To join a sample session, click the Join a Default Session button. You'll be asked to enter your Name and provide a password (enter mp for moderator password, vp for viewer password).


To schedule your own session, click the Schedule a Session button. Enter admin@test.com for username and admin for password.


This virtual machine is also a full BigBlueButton development enviornment that you use to checkout, modify the source, and build you own version of BigBlueButton.
